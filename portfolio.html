Lambda Group Organizer

Tech Stack:

React, Google Firestore, Semantic UI


Description: Lambda Group Organizer started as a winning 2 day hackathon project that Lambda School requested we make production ready, which took a month to get to the beta version. This application fixes Lambda Schools trouble with signing students up for build weeks. Previously they were using Google sheets in order to sign students up to projects. Now students can sign themselves up with minimal fuss and hassle for everyone involved!


The application does live updates using websockets (through firebase). It also allows the administrator to add an entire build week using a csv, and then customize that build week as the signup process moves along.

MoodiBeats

Tech Stack:

React.js, Node.js, SQLite(locally), PostGreSQL(production), Youtube API, Data Science Side API from a Django RESTful api.


Description:


MoodiBeats is an app designed for content creates who want royalty free music that is actually free to use. You can make a playlist based on the mood you are looking for and the music is updated daily.

This application was created by a team of 5 web developers and 2 data scientists over the course of 5 weeks.

Media Masters

Tech Stack:

Wordpress, Divi Theme, hand customized HTML/CSS/JavaScript


Description: Media Masters is a small videography business in Chantilly Virginia owned by Abby Sternburg. The website was created in 1994, but by 2018 Abby was ready for a design refresh. She requested that I keep almost all of the current content but wanted a new look.


Abby wanted a website that would be easy to update and change herself. This website was created using wordpress with a drag and drop builder placed on top to make things easy for her to update and change in the future, with some hand coded custom sections.

Conway's Game of Life

Tech Stack:

React.js


Description: My second ever game application! I have had enough people enjoy playing it, I thought I would share even though the styling isn't quite finished. The game is a zero-player game, meaning that its evolution is determined by its initial state, requiring no further input. One interacts with the Game of Life by creating an initial configuration and observing how it evolves, or, for advanced players, by creating patterns with particular properties.


Keep in mind, this one is still in it's UI polishing phase, but it's a great example of what you can do with some computer science knowledge


Enjoy!
